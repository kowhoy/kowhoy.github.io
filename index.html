Scala函数柯里化(Currying)

Words count 389 Reading time 1 mins.

  • Scala

将函数进行往下分解

def add(x:Int, y:Int) = x + y

// ==> 柯里化

def add(x:Int)(y:Int) = x+y
// => 转化
def add(x:Int)=(y:Int) => x + y

转化过程： 将add函数看成 x = y的一个匿名函数 x+y
接受一个x的参数，返回一个匿名函数，这个匿名函数相当与一个闭包

柯里化的意义:将多参数的函数等价转化为多个单参数的函数，所有的函数都统一了，?方便lambda演算?
