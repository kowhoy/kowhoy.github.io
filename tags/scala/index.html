Scala函数柯里化(Currying)

Words count 389 Reading time 1 mins.

  • Scala

将函数进行往下分解

def add(x:Int, y:Int) = x + y

// ==> 柯里化

def add(x:Int)(y:Int) = x+y
// => 转化
def add(x:Int)=(y:Int) => x + y

转化过程： 将add函数看成 x = y的一个匿名函数 x+y
接受一个x的参数，返回一个匿名函数，这个匿名函数相当与一个闭包

柯里化的意义:将多参数的函数等价转化为多个单参数的函数，所有的函数都统一了，?方便lambda演算?

Scala传值函数和传名函数

Words count 831 Reading time 1 mins.

  • Scala

Scala解释器在解析函数参数(function arguments)有两种方式:

  1. 传值调用(call-by-value)
    先计算参数表达式的值，再应用到函数内部
  2. 传名字调用(call-by-name)
    将未计算的表达式直接应用到函数内部

两种函数的区别

  1. 传值调用
    这种方式，在进入函数体之前就已经将表达式进了计算，在函数内部如果多次使用也不会重复计算，提高了效率
  2. 传名调用
    如果函数体内部没有使用到该参数，就不会进行计算，这样看效率会更高
