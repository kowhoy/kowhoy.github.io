Scala字符串

  1. 单双引号的区别
    双引号的类型为String, 单引号的类型为Char

示例：

在对字符串进行filter操作，应使用单引号，使用双引号会warning，且结果非预期

scala> val str = "wojiushiwoa"
str: String = wojiushiwoa

scala> val new_str = str.filter(_ != 'a')
new_str: String = wojiushiwo

scala> val new_str_2 = str.filter(_ != "a")
                                    ^
       warning: comparing values of types Char and String using `!=` will always yield true
new_str_2: String = wojiushiwoa
  1. drop() 和 take()
val str = "wojiushiwo"

val droped_str = str.drop(2)
droped_str: String = jiushiwo

val taked_str = droped_str.take(3)
taked_str: String = jiu

"woJiushiWo".drop(1).take(8).capitalize
// String = OJiushiW

⚠️：scala 和 python 中的captialize有区别，python中是将首字母大写，其余字母统一小写，scala中只是将首字母大写。

  1. 字符串的比较
    scala中使用==进行比较字符串是否相同，在比较之前，会先进行比较两方是否为null，所有将字符串与null进行比较是不会出错的。
scala> val s1 = "hey"
s1: String = hey

scala> val s2 = "hey"
s2: String = hey

scala> val s3 = "Hey"
s3: String = Hey

scala> val s4 = null
s4: Null = null

scala> s1 == s2
res1: Boolean = true

scala> s1 == s4
res2: Boolean = false

scala> s1.toUpperCase == s3.toUpperCase
res3: Boolean = true

scala> s1.equalsIgnoreCase(s3)
res4: Boolean = true

scala> s1.toUpperCase == s4.toUpperCase
                            ^
       error: value toUpperCase is not a member of Null

scala> s4.equalsIgnoreCase(s1)
          ^
       error: value equalsIgnoreCase is not a member of Null

scala> s1.equalsIgnoreCase(s4)
res7: Boolean = false
  1. 多行字符串及stripMargin()的使用
    scala的多行字符串很很多其他语言一致，使用"""...."""三个双引号包裹，stripMargin是为了解决多行双引号的对齐问题，可以在多行字符串行首加入默认的|作为连接符

示例:

scala> val sql_str = """
     |  |select count(*)
     | |from order
     |   |where 1=1
     | """
sql_str: String =
"
 |select count(*)
|from order
  |where 1=1
"

scala> sql_str.stripMargin('|')
res10: String =
"
select count(*)
from order
where 1=1
"

scala> sql_str.stripMargin
res11: String =
"
select count(*)
from order
where 1=1
"

可以使用 replaceAll(“\n”, “ “) 将多行字符串转化为一行

sql_str.stripMargin.replaceAll("\n", " ")
res12: String = " select count(*) from order where 1=1 "
  1. 字符串分割
    ⚠️：需要对空字符的处理
scala> val s = "eggs, milk, butter, Coco Puffs"
s: String = eggs, milk, butter, Coco Puffs

scala> val a = s.split(",")
a: Array[String] = Array(eggs, " milk", " butter", " Coco Puffs")

scala> val a = s.split(',')
a: Array[String] = Array(eggs, " milk", " butter", " Coco Puffs")

scala> a.foreach(println)
eggs
 milk
 butter
 Coco Puffs

scala> val b = s.split(',').map(_.trim)
b: Array[String] = Array(eggs, milk, butter, Coco Puffs)

scala> b.foreach(println)
eggs
milk
butter
Coco Puffs

scala中split方法，对于不同类型(char/String)类型的参数，使用的是不同的方法，但是对结果没有影响。

  1. 变量替换
    在字符串前面加s 变量名称前面加$可进行变量替换

示例：

scala> val name = "koko"
name: String = koko

scala> val age = "21"
age: String = 21

scala> val weight = "62"
weight: String = 62

scala> s"$name is $age years old, and weights $weight kg"
res15: String = koko is 21 years old, and weights 62 kg

表达式使用 ${}

示例:

scala> s"Age next year: ${age.toInt + 1}"
res17: String = Age next year: 22

如果使用对象字段也需要{}

scala> case class Student(name:String, score:Int)
defined class Student

scala> val ko = Student("ko", 149)
ko: Student = Student(ko,149)

scala> s"${ko.name} has a score of ${ko.score}"
res19: String = ko has a score of 149

在字符串前面加f可以格式化字符串

示例:

scala> val weight = 62.595
weight: Double = 62.595

scala> f"$name is $age years old, and weights $weight%.2f kg"
res29: String = koko is 21 years old, and weights 62.60 kg

在字符串前面加 raw插入符，不会对字符串进行转义

示例:

scala> raw"foo\nraw"
res30: String = foo\nraw

scala> "foo\nraw"
res31: String =
foo
raw
  1. 正则
    字符串后面接.r变成Regex类正则表达式

使用 findFirstIn 或 findAllIn 匹配

使用 replaceFirstIn 或 replaceAllIn 替换

示例:

scala> val pattern = "Scala".r
pattern: scala.util.matching.Regex = Scala

scala> val str = "Scala is Sca Scala SSS"
str: String = Scala is Sca Scala SSS

scala> pattern.findFirstIn(str).foreach(println)
Scala

scala> pattern.findAllIn(str).foreach(println)
Scala
Scala

scala> pattern.replaceFirstIn(str, "python")
res35: String = python is Sca Scala SSS

scala> pattern.replaceAllIn(str, "python")
res36: String = python is Sca python SSS
